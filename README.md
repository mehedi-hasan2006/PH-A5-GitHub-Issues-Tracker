1️⃣ What is the difference between var, let, and const?

    Ans: 

    var: var is a function scope. It can be redeclare & reassign. 
    
    let : let is a block scope. It can't be redeclare. 
    
    const: const also block scope but it's value can't be changed, redeclare or reassign.

2️⃣ What is the spread operator (...)?

    Ans: spread operator(...) is used to expand elements of an array & object. It copies arrays and objects.

3️⃣ What is the difference between map(), filter(), and forEach()?

    Ans: 
    
    map(): It's perform  operation of every  array  elements and return a new array.

    filter(): It's return a new array based on a condition.

    forEach(): It's perform a loop operation of evey array elements but it's doesn't return anything.  

4️⃣ What is an arrow function?

    Ans: An arrow function is a shorter way to write a function in JavaScript using the => syntax. 

5️⃣ What are template literals?
